This is a responsive, single-page portfolio website designed to highlight my work and provide a way for potential clients or employers to get in touch.

**Live Preview:** Open `index.html` in your web browser.

## 🛠️ Technologies Used

-   **HTML5:** Semantic structure and content.
-   **CSS3:** Custom styling with responsive design, CSS variables, and animations.

-   **FontAwesome:** Scalable vector icons for a professional look.
